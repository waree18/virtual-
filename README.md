# virtual-

## การพัฒนาเว็บแอปพลิเคชันพัฒนาห้องเรียนเสมือนจริง 360 องศา  
### กรณีศึกษา แผนกวิชาเทคโนโลยีสารสนเทศ วิทยาลัยอาชีวศึกษาภูเก็ต

---

## 📌 บทคัดย่อโครงงาน

โครงงานนี้เป็นการพัฒนาเว็บแอปพลิเคชันห้องเรียนเสมือนจริง 360 องศา  
เพื่อเพิ่มประสิทธิภาพในการจัดการเรียนการสอน และส่งเสริมการเรียนรู้แบบดิจิทัล  
โดยมุ่งเน้นการสร้างประสบการณ์เสมือนจริง (Virtual Experience)  
ให้กับผู้เรียนในแผนกวิชาเทคโนโลยีสารสนเทศ วิทยาลัยอาชีวศึกษาภูเก็ต

---

## 🎯 วัตถุประสงค์ของโครงการ

- เพื่อพัฒนาเว็บแอปพลิเคชันห้องเรียนเสมือนจริง 360 องศา
- เพื่อศึกษาความพึงพอใจของผู้ใช้งานระบบ
- เพื่อส่งเสริมการเรียนรู้ผ่านเทคโนโลยี VR และสื่ออินเทอร์แอคทีฟ

---

## 💻 เทคโนโลยีที่ใช้

- Node.js
- Express.js
- HTML5 / CSS3
- JavaScript
- 360° Panorama Technology
- VR Support

---

## 🧩 คุณสมบัติของระบบ

- แสดงภาพห้องเรียนแบบ 360 องศา
- รองรับการใช้งานผ่านเว็บเบราว์เซอร์
- รองรับโหมด VR
- ระบบจัดการข้อมูลผ่าน Server
